弹性布局概述

弹性布局介绍

网页布局的传统解决方案，基于盒子模型 + 浮动 + 定位实现。它对于那些特殊布局非常不方便，比如，垂直居中就不容易实现。

2009年，W3C提出了一种新的方案—Flex布局(弹性布局)，可以简便、完整、响应式地实现各种页面布局。现在，它已经得到了所有浏览器的支持，这意味着，现在就能很安全地使用这项功能。支持 Flex 布局的浏览器

弹性布局已经成为布局的首选方案，本章介绍弹性布局的使用方式。

弹性布局概念

Flex 布局是在 CSS3 中引入的，又称为弹性盒模型。该模型决定一个盒子在其他盒子中的分布方式以及如何处理可用的空间。

Flex 布局对于设计比较复杂的页面非常有用，可以轻松实现屏幕和浏览器窗口大小发生变化时保持元素的相对位置和大小不变，同时减少了依赖浮动布局实现元素位置的定义以及重置元素的大小。

Flex布局在定义伸缩项目大小时伸缩容器会预留一些可用空间，可以调节伸缩项目的相对大小和位置。例如，可以确保伸缩容器中的多余空间平均分配给多个伸缩项目。当然，如果伸缩容器没有足够大的空间放置伸缩项目，浏览器会根据一定的比例减少伸缩项目的大小，使其不溢出伸缩容器。

综合而言，Flex布局主要具体有以下几点功能：

  • 在屏幕和浏览器窗口大小发生改变时也可以灵活地调整布局。
  • 控制元素在页面的布局方向。
  • 按照不同于文档流的排序方式对屏幕上的元素重新排序。

如何在网页中使用弹性盒模型呢？要开启弹性盒模型，只需要设置拥有子盒子的 display 属性值为 flex 即可。
采用 Flex 布局的元素，称为 Flex 容器（flex container），简称”容器”。它的所有子元素自动成为容器成员，称为Flex 项目（flex item），简称”项目”。

快速体验

我们在弹性布局介绍这一小结中说过传统的布局方式不容易实现，垂直居中。那下面我们使用弹性布局演示下如何实现水平和垂直居中，你需要观察效果即可，这里用到的属性后续会逐个介绍。

首先创建一个div，嵌套一个span，弹性布局实现水平和垂直居中的话内部是行内元素也可以实现。结构代码如下：

<div class="box">
  <span>Hello Flex</span>
</div>

接下来设置外层盒子的大小，CSS代码如下：

.box {
  width: 300px;
  height: 300px;
}

接下来使用弹性布局实现盒子内部的span水平和垂直居中的效果，CSS代码如下：

.box {
  width: 300px;
  height: 300px;
  /* 设置盒子的布局方式为弹性布局 */
  display: flex;
  /* 设置项目水平居中 */
  justify-content: center;
  /* 设置项目垂直居中 */
  align-items: center;
}

使用 flex 布局首先给父元素设置display:flex;。
注意：设为 flex 布局以后，子元素的 float、clear 和 vertical-align 属性将失效，如果子元素是 span 等行内元素的话此时这些元素也可以设置大小，类似于行内块元素。

弹性布局常用属性

在弹性布局中可以分别给 **Flex容器和Flex项目 **分别设置不同的属性。

  • 下面列出的是Flex容器的属性：
    • flex-direction：设置主轴的方向（即项目的排列方向）
    • justify-content：定义了项目在主轴上的对齐方式。
    • align-items：定义项目在交叉轴上如何对齐。
    • flex-wrap：默认情况下，项目都排在一条线（又称轴线）上。flex-wrap属性定义，如果一条轴线排不下，如何换行。
    • flex-flow：flex-direction属性和flex-wrap属性的简写形式，默认值为row nowrap。
    • align-content：定义了多根轴线的对齐方式。如果项目只有一根轴线，该属性不起作用。
  • 下面列出的是Flex项目的属性：
    • order：定义项目的排列顺序。数值越小，排列越靠前，默认为0。
    • flex-grow：定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大。如果所有项目的flex-grow属性都为1，则它们将等分剩余空间（如果有的话）。如果一个项目的flex-grow属性为2，其他项目都为1，则前者占据的剩余空间将比其他项多一倍。
    • flex-shrink：定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。
    • flex-basis：定义了在分配多余空间之前，项目占据的主轴空间。浏览器根据这个属性，计算主轴是否有多余空间。它的默认值为auto，即项目的本来大小。
    • flex：该属性是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选。
    • align-self：允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。

接下来我们分别来介绍和演示常用的属性。

Flex 容器属性

在学习 Flex 容器属性之前我们需要再来了解 Flex 容器的两个概念：主轴和交叉轴。容器默认存在两根轴：水平的主轴和垂直的交叉轴。项目默认沿主轴排列。主轴和交叉轴不是固定的，可以通过 flex-direction 属性来改变。
容器属性总共有六个，这里重点掌握前四个，分别是：flex-direction、justify-content、align-items、flex-wrap。

flex-direction

flex-direction 属性决定主轴的方向（即项目的排列方向），该属性的语法如下：

.box {
  flex-direction: column | column-reverse | row | row-reverse;
}

flex-direction属性的取值有4个，每个值及解释：

  • column：主轴为垂直方向，起点在上沿。
  • column-reverse：主轴为垂直方向，起点在下沿。
  • row（默认值）：主轴为水平方向，起点在左端。
  • row-reverse：主轴为水平方向，起点在右端。

每个属性的演示效果如图所示

下面我们来演示下 flex-direction 的使用，通过flex-direction可以方便排列子项目的排列方式，我们通过 flex 的方式让 ul 里的 li 横向排列，页面的结构代码如下：

<ul>
  <li>1</li>
  <li>2</li>
  <li>3</li>
</ul>

设置li初始的样式，去除li前面的黑点，设置li的宽度和背景颜色，代码如下：

li {
  list-style-type: none;
  width: 50px;
  background-color: bisque;
}

在使用flex布局之前首先要把父元素设置为flex的容器，所以这里给ul设置display: flex; 代码如下：

ul {
  display: flex;
}

页面运行的效果如图所示。

通过图可以看到li已经横向展示，这里我们没有设置 flex-direction，该属默认值是row，也就是在主轴(当前是水平方向)上排列flex项目(当前是li)。
接下来把容器的flex-direction属性设置为row-reverse观察运行效果，代码如下：

ul {
  display: flex;
  flex-direction: row-reverse;
}

运行效果如所示。

通过图2-5我们可以看到flex-direction的值为row-reverse的时候是从最右端开始排列flex项目，也就是从主轴的方向是从右向左。
当flex-direction的值为column的时候，主轴的方向改变为垂直方向，交叉轴改变为水平方向。接下来你可以尝试把flex-direction改变为其它两个值观察效果。

justify-content

justify-content 属性定义了项目(子元素)在主轴上的对齐方式，该属性的语法如下：

.box {
  justify-content: flex-start | flex-end | center | space-between | space-around;
}

它可能取5个值，具体对齐方式与轴的方向有关。下面假设主轴为从左到右。

  • flex-start（默认值）：主轴起点对其
  • flex-end：主轴终点对其
  • center： 居中
  • space-between：两端对齐，项目之间的间隔都相等。
  • space-around：每个项目两侧的间隔相等。所以，项目之间的间隔比项目与边框的间隔大一倍。

通过图所示你可以清晰的看到该属性的不同取值时子元素的排列方向。

align-items

align-items 属性定义项目在交叉轴上如何对齐，该属性语法如下：

.box {
  align-items: flex-start | flex-end | center | baseline | stretch;
}

它可能取5个值。具体的对齐方式与交叉轴的方向有关，下面假设交叉轴从上到下。

  • flex-start：交叉轴的起点对齐。
  • flex-end：交叉轴的终点对齐。
  • center：交叉轴的中点对齐。
  • baseline: 项目的第一行文字的基线对齐。
  • stretch（默认值）：如果项目未设置高度或设为auto，将占满整个容器的高度。

通过图所示你可以清晰的看到该属性的不同取值时子元素的排列方向。

flex-wrap

默认情况下，项目都排在一条线（又称”轴线”）上。flex-wrap属性定义，如果一条轴线排不下，如何换行，该属性语法如下：

.box{
  flex-wrap: nowrap | wrap | wrap-reverse;
}

三个取值的含义如下：

  • nowrap(默认值)：不换行，如图所示。

  • wrap：换行，第一行在上方，如图所示。

  • wrap-reverse：换行，第一行在下方，如图所示。

flex-flow

flex-flow 属性是 flex-direction 属性和 flex-wrap 属性的简写形式，默认值为 row nowrap，该属性语法如下：

.box {
  flex-flow: <flex-direction> || <flex-wrap>;
}

align-content

align-content 属性定义了多根轴线的对齐方式。如果项目只有一根轴线，该属性不起作用。什么时候会有多根轴线，当 flex-wrap 有换行的时候的时候会产生多根轴线。该属性语法如下：

.box {
  align-content: flex-start | flex-end | center | space-between | space-around | stretch;
}

它可能取6个值。具体的对齐方式与交叉轴的方向有关，下面假设交叉轴从上到下。

  • flex-start：与交叉轴的起点对齐。
  • flex-end：与交叉轴的终点对齐。
  • center：与交叉轴的中点对齐。
  • space-between：与交叉轴两端对齐，轴线之间的间隔平均分布。
  • space-around：每根轴线两侧的间隔都相等。所以，轴线之间的间隔比轴线与边框的间隔大一倍。
  • stretch（默认值）：轴线占满整个交叉轴。

通过图所示你可以清晰的看到该属性的不同取值时子元素的排列方向。

Flex 项目属性

Flex项目有6个属性，分别是：

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • align-self

下面我们依次来解释，这里重点掌握两个属性分别是：order 和 flex。

order

order属性定义项目的排列顺序。数值越小，排列越靠前，默认为0，该属性语法如下：

.item {  order: <integer>; }

下图演示该属性为不同值时项目的排列。

align-self

align-self属性允许单个项目在交叉轴上有与其他项目不一样的对齐方式，可覆盖align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素，则等同于stretch，该属性语法如下：

.item {  align-self: auto | flex-start | flex-end | center | baseline | stretch; }

该属性可能取6个值，除了auto，其他都与align-items属性完全一致，演示效果如图所示。

flex-grow

flex-grow属性定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大，该属性语法如下：

.item {   flex-grow: <number>; /* default 0 */ }

如果所有项目的flex-grow属性都为1，则它们将等分剩余空间（如果有的话）。如果一个项目的flex-grow属性为2，其他项目都为1，则前者占据的剩余空间将比其他项多一倍。效果如图所示。

flex-shrink

flex-shrink属性定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小，该属性语法如下：

.item {   flex-shrink: <number>; /* default 1 */ }

如果所有项目的flex-shrink属性都为1，当空间不足时，都将等比例缩小。如果一个项目的flex-shrink属性为0，其他项目都为1，则空间不足时，前者不缩小。负值对该属性无效，效果如图所示。

flex-basis

flex-basis 指定了 flex 元素在主轴方向上的初始大小。该属性语法如下：

.item {   flex-basis: <length> | auto; /* default auto */ }

当一个元素同时被设置了 flex-basis (除值为 auto 外) 和 width (或者在 flex-direction: column 情况下设置了height) , flex-basis 具有更高的优先级.

flex

flex属性是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选，该属性语法如下：

.item {   flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ] }

该属性有两个快捷值：auto (1 1 auto) 和 none (0 0 auto)。
建议优先使用这个属性，而不是单独写三个分离的属性，因为浏览器会推算相关值。

案例2：网页布局

案例介绍

训练要点

  • 使用弹性盒模型排版网页元素。

需求说明
制作如图所示的优酷友情链接页面，要求如下：

  • 使用HTML5结构元素布局页面结构。
  • 使用弹性盒模型布局网页头部的内容，主轴的对齐方式为space-around，侧轴的对齐方式为center.
  • 使用弹性盒模型布局网页主题部分的内容，主轴的对齐方式为center。左边“公司介绍”模块的伸缩性为1，并且它自身也是一个伸缩容器，伸缩流方向为column。右边模块的伸缩性为4.
  • 主体部分的右侧图片整体模块是一个弹性盒子，伸缩换行为wrap，主轴的对齐方式为flex-start。
  • 使用弹性盒模型布局网页的尾部内容，伸缩流方向为column，主轴对齐方式和侧轴的对齐方式都是center。
