2D 转换

2D 转换是改变标签在二维平面上的位置和形状，使用的属性是 transform，翻译成汉语具有”变换”或者”改变”的意思。通过此属性具有非常强大的功能，比如可以实现元素**位移 translate、旋转 rotate、缩放 scale **等效果。

位移

translate 可以在水平或垂直方向移动元素

使用方法

transform: translate(x, y)
transform: translateX(n)
transfrom: translateY(n)
  • x, y, n 可以是像素值，可以是百分比
  • 如果是百分比是指盒子自身大小的百分之多少

注意点

  • 2D 的移动主要是指 水平、垂直方向上的移动
  • translate 不影响其他元素的位置
  • translate 中的 100%单位，是相对于本身的宽度和高度来进行计算的
  • 对行内元素无效

案例：盒子水平垂直居中

.parent {
  width: 500px;
  height: 500px;
  border: 1px solid red;
  margin: 50px auto;
  position: relative;
}

.child {
  width: 100px;
  height: 100px;
  background-color: blue;
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

旋转

rotate 旋转 可以让元素在二维平面内顺时针或者逆时针旋转

使用方法

transform: rotate(度数);
  • rotate 中的度数，单位是 deg
  • 角度为正时：顺时针旋转，角度为负时：逆时针旋转
  • 默认旋转的中心点是元素的中心点

代码演示

  • 基本代码演示
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  margin: 100px auto;
  transition: transform 2s;
}

.box:hover {
  transform: rotate(360deg);
}
  • 旋转的笑脸
img {
  transition: transform 2s;
}

img:hover {
  transform: rotate(360deg);
}

注意：旋转的中心点是元素的中心点

案例：旋转三角

<div class="box">
  <span class="sj"></span>
</div>
* {
  margin: 01;
  padding: 0;
  box-sizing: border-box;
}

.box {
  position: relative;
  width: 250px;
  height: 30px;
  border: 1px solid #000;
  margin: 30px 100px;
}

.sj {
  position: absolute;
  /* transform 对行内元素无效，转换成行内块元素 */
  display: inline-block;
  width: 8px;
  height: 8px;
  right: 10px;
  top: 50%;
  border-top: 1px solid #000;
  border-right: 1px solid #000;
  transform: translateY(-50%) rotate(135deg);
  transition: transform 1s;
}

.box:hover .sj {
  transform: translateY(-50%) rotate(315deg);
}

改变旋转中心点

  • 可以使用 transform-origin 改变旋转的中心点
  • transform-origin 基础语法
transform-origin: x y;
  • 注意后面的参数 x 和 y 用空格隔开
  • x y 默认旋转的中心点是元素的中心 (50% 50%)，等价于 center  center
  • 还可以给 x y 设置像素或者方位名词(top、bottom、left、right、center)

案例：旋转美女图片

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

ul {
  list-style: none;
}

ul li {
  float: left;
  overflow: hidden;
  width: 300px;
  height: 200px;
  border: 1px solid #000;
  margin-right: 10px;
}

ul li img {
  width: 100%;
  height: 100%;
  transform-origin: left bottom;
  transform: rotate(90deg);
  transition: transform 1s;
}

ul li:hover img {
  transform: rotate(0deg);
}

缩放

scale 缩放，用来控制元素的放大与缩小。缩放和旋转一样，默认都是以自己的中心点进行变化的。

使用方法

transform: scale(x, y);
  • 注意，x 与 y 之间使用逗号进行分隔
  • transform: scale(1, 1): 宽高都放大一倍，相当于没有放大
  • transform: scale(2): 如果只写了一个参数，第二个参数就和第一个参数一致
  • transform:scale(0.5, 0.5): 缩小
  • scale 最大的优势：可以设置转换中心点缩放，默认以中心点缩放，而且不影响其他盒子

代码演示

img {
  transition: transform 1s;

  /* 改变缩放的中心点 */
  /* transform-origin: left top; */
  /* transform-origin: left center; */
  transform-origin: right top;
}

img:hover {
  /* 默认以中心点变化 */
  /* transform: scale(1.2);
  transform: scale(2, 2); */
  transform: scale(1.2);
}

案例：缩放美女图片

实现类似于优酷的鼠标放到图片上放大的效果

ul li img {
  width: 100%;
  height: 100%;
  transition: transform 1s;
}

ul li:hover img {
  transform: scale(1.2);
}

transform 综合写法

使用方式

transform 可以同时实现多个转换效果，使用方式如下：

transform: translate(x, y) rotate(度数) scale(x, y);

代码演示

让 div 同时在 5s 内实现位移、旋转、缩放的效果

transform: translateX(200px) rotate(360deg) scale(0.8);

顺序问题

注意 transform 在使用多个值的时候有顺序问题，例如上例子中把位移和渲染交换顺序执行的效果完全不一样

transform: rotate(360deg) translateX(200px) scale(0.8);

这是因为：元素旋转之后坐标系也跟着旋转

可执行动画的属性

网页制作者可以定义哪个属性以哪种方式动画，这样能制作出复杂的过渡。因为有些属性动画无意义，所以 可动画属性列表 是一个有限集合。

CSS3 关键帧动画(animation)

过渡只能定义首尾两个状态，关键帧动画可以定义多个状态，或者用关键帧的话来说，过渡动画只能定义第一帧和最后一帧这两个关键帧，而关键帧动画则可以定义任意多的关键帧，因而能实现更复杂的动画效果。

二者相同点：都是随着时间改变元素的属性值。

过渡与动画不同点：transition 需要触发一个事件(hover 事件或 click 事件等)才会随时间改变其 css 属性；而 animation 在不需要触发任何事件的情况下也可以显式的随着时间变化来改变元素 css 的属性值，从而达到一种动画的效果

animation 属性用来指定一组或多组动画，每组之间用逗号相隔。

使用方式

定义动画序列

@keyframes 动画名称 {
  from {
    width: 100px;
  }
  to {
    width: 200px;
  }
}

使用动画

.box {
  /* 调用动画 */
  animation-name: 动画名称;
  /* 持续时间 */
  animation-duration: 持续时间;
}

代码演示

.box {
  width: 100px;
  height: 30px;
  background-color: red;
  animation-name: long;
  animation-duration: 2s;
  /* 动画执行完成之后保持位置 */
  animation-fill-mode: forwards;
}

@keyframes long {
  from {
    width: 100px;
  }

  to {
    width: 500px;
  }
}

案例：珊珊来迟的美女

<div class="box">
  <img src="./images/mv1.jpg" alt="" />
</div>
.box {
  width: 300px;
  height: 200px;
  border: 1px solid black;
}

.box img {
  width: 100%;
  height: 100%;
  transform: translateX(-100%);
  animation-name: move;
  animation-duration: 2s;
  animation-fill-mode: forwards;
}

@keyframes move {
  from {
    transform: translateX(-100%);
  }

  to {
    transform: translateX(0);
  }
}

定义动画序列的另一种方式

from 和 to 类似于过渡动画，只能定义动画开始和结束的位置。关键帧动画可以通过百分数来定义，动画间隔时间的每一个位置处的动画帧，这个百分数是时间间隔的百分数

@keyframes move {
}
  • from 和 0% 代表第一帧
  • to 和 100% 代表最后一帧
  • 50% 代表时间间隔到达一半的时候

案例：走矩形的盒子

.box {
  width: 100px;
  height: 100px;
  background-color: blue;
  animation-name: move;
  animation-duration: 10s;
}

@keyframes move {
  /* 第0s的位置 */
  0% {
    transform: translate(0);
  }

  /* 第 25% * 10s 的位置 */
  25% {
    transform: translate(600px, 0);
  }

  /* 第 50% * 10s 的位置 */
  50% {
    transform: translate(600px, 500px);
  }

  /* 第 75% * 10s 的位置 */
  75% {
    transform: translate(0, 500px);
  }

  /* 第 100% * 10s 的位置 */
  100% {
    transform: translate(0);
  }
}

animation 常用属性

属性名称 描述
@keyframes 定义动画序列
animation 复合属性
animation-name 动画序列的名称，必须
animation-duration 动画序列完成的时间，秒或毫秒，必须
animation-timing-function 一个动画周期执行的节奏，默认 ease
animation-delay 关键帧动画开始时间，默认 0
animation-iteration-count 关键帧动画执行次数，默认 1，infinite 无穷次
animation-direction normal 正常, alternate 逆向播放
animation-play-state 动画运行 running 默认，或者暂停 paused
animation-fill-mode 动画结束后的状态，保持 forwards，回到起始位置 backwards 默认。
  • 代码演示
.box {
  width: 100px;
  height: 100px;
  background-color: blue;
  /* 动画名称 */
  animation-name: move;
  /* 动画花费时长 */
  animation-duration: 2s;
  /* 动画速度曲线 */
  animation-timing-function: ease-in-out;
  /* 动画等待多长时间执行 */
  animation-delay: 2s;
  /* 规定动画播放次数 infinite: 无限循环 */
  animation-iteration-count: infinite;
  /* 是否逆行播放 */
  animation-direction: alternate;
  /* 动画结束之后的状态 */
  animation-fill-mode: forwards;
}

div:hover {
  /* 规定动画是否暂停或者播放 */
  animation-play-state: paused;
}

animation 复合属性

animation: name duration timing-function delay iteration-count direction
  fill-mode;
  • 简写属性里面不包含 animation-paly-state
  • 暂停动画 animation-paly-state: paused，经常和鼠标经过等其他配合使用
  • 要想动画走回来，而不是直接调回来：animation-direction: alternate
  • 盒子动画结束后，停在结束位置：animation-fill-mode: forwards
animation: move 2s linear 1s infinite alternate forwards;

动画执行节奏

animation-timing-function: 设置动画的执行节奏，默认是 ease。
这个属性的值和过渡动画的时候一样，但是多了一个 steps()

属性值 描述
linear 匀速执行
ease 默认，慢速开始慢速结束
ease-in 先慢后快
ease-out 先快后慢
ease-in-out 先慢后快再慢
steps() 指定事件函数中的间隔数量(步长)，几次执行完 
.box {
  width: 0px;
  height: 50px;
  background-color: blue;
  /* 10次执行完毕 */
  animation: move 5s steps(10) forwards;
}

@keyframes move {
  0% {
    width: 0px;
  }

  100% {
    width: 400px;
  }
}

3D 转换

3D 转换在 2D 转换的基础上多了一个 Z 轴，可以实现立体的效果(近大远小)


https://www.html5tricks.com/demo/html5-3d-game-scene/index.html

https://www.html5tricks.com/demo/html5-3d-parallax-photos/index.html

3D 转换同样包含：位移、旋转和缩放，在位移、旋转、缩放的过程中可以和 2D 转换一样沿着 X、Y 轴进行，在 3D 转换中还可以沿着 Z 轴进行。

3D 坐标系：

位移

3D 移动就是在 2D 移动的基础上多加了一个可以移动的方向，就是 z 轴方向

使用方法

/* 沿X、Y轴方向移动和2D转换一样 */
transform: translateX(100px);
transform: translateY(100px);

/* 沿Z轴方向移动 */
transform: translateZ(100px);
/* XYZ三个轴共同作用 */
transform: translate3d(100px, 200px, 200px);
/* 等同上面效果 */
transform: translateX(100px) translateY(200px) translateZ(200px);

你会发现设置 translateZ() 按 Z 轴平移不起作用，这是因为在使用 3D 转换之前首先要开启透视，下一小节见。

透视 perspective

概念介绍

在网页中实现 3D 效果首先要了解透视，也可以称为视距，指人眼到屏幕的距离。视距越大在屏幕上显示的画面越大，视距越小在屏幕上显示的画面越小。

当为元素定义透视 perspective 属性时，其子元素会获得透视效果，而不是元素本身，透视的单位是像素。
在开启 3D 效果之前，首先要开启透视

使用方法

  • 给父元素设置透视 perspective
  • Z 轴的偏移不能能和透视的距离一样，否则无意义(人眼和物体的距离相当)
<div class="parent">
  <div class="child"></div>
</div>
.parent {
  width: 400px;
  height: 400px;
  margin: 100px auto;
  border: 1px solid #000;
  /* 设置透视(视距) */
  perspective: 500px;
}

.child {
  width: 200px;
  height: 200px;
  background-color: red;
  margin: 100px;
  /* transform: translateZ(200px); */
  /* transform: translateX(100px) translateY(100px) translateZ(100px); */
  /* transform: translate3d(100px, 100px, 100px); */
  transform: translate3d(100px, 100px, -100px);
}

观察 Z 轴的偏移

为了证明 Z 轴的存在，让盒子沿着 Y 轴旋转，通过开发人员工具改变 Y 轴旋转角度，观察 Z 轴偏移。

在上一小节的案例基础上，给父元素设置背景颜色和 Y 轴旋转

background-color: blue;
transform: rotateY(0deg);

子元素设置沿 Z 轴偏移

transform: translateZ(100px);

改变父元素沿 Y 轴旋转角度，观察到的结果

观察发现，当父元素沿 Y 轴旋转，子元素也会沿 Y 轴旋转，但是没有看到子元素和父元素之间 Z 轴上的偏移。这里是有问题的，根之前描述的观点是冲突的。

要解决这种冲突我们就需要了解另一个属性 transform-style

transform-style

transform-style 是设置转换的过程中，子元素是是位于 3D 空间中还是平面中，这个属性有两个值：

  • flat：默认值，设置元素的子元素位于该元素的平面中。
  • preserve-3d：元素的子元素应位于 3D 空间中。

在上一小节中，因为父元素旋转但是子元素的渲染是处于父元素的二维平面空间中(transform-style 的值默认是 flat)，所以没有观察到 Z 轴的偏移。

那么我们只需要把父元素的 transform-style 的值重新改成 preserve-3d 就可以观察到 Z 轴的偏移。

修改上例给父元素增加 transform-style 属性为 preserve-3d，让子元素在 3D 空间渲染。

注意：transform-style 设置给需要需要 3D 渲染空间元素的直接父元素上。

transform-style: preserve-3d;

旋转

3D 旋转在 2D 旋转的基础上除了增加沿着 Z 轴进行旋转以外，还可以沿着自定义轴进行旋转，同时增加了 3D 效果（近大远小）

使用方法

/* 绕X轴旋转45度 */
transform: rotateX(45deg);
/* 绕Y轴旋转45度 */
transform: rotateY(45deg);
/* 绕Z轴旋转45度 */
transform: rotateZ(45deg);
/* 绕自定义轴旋转45度 */
transform: rotate3d(x, y, z, 45deg);

旋转方向的识别，左手定律，左手握住坐标轴大拇指指向坐标轴正方向，四指弯曲的方向是旋转的正方向。

代码演示-X 轴旋转

.parent {
  width: 400px;
  height: 400px;
  margin: 100px auto;
  border: 1px solid #000;
  /* 设置透视(视距) */
  perspective: 500px;
}

.child {
  width: 200px;
  height: 200px;
  background-color: red;
  margin: 100px;
  /* 设置过渡动画 */
  transition: transform 2s;
}

.parent:hover .child {
  /* 切换正负值，观察旋转方向 */
  /* transform: rotateX(45deg); */
  transform: rotateX(45deg);
}

代码演示-Y 轴旋转

/* 切换正负值，观察旋转方向 */
transform: rotateY(45deg);
/* transform: rotateY(-45deg); */

代码演示-Z 轴旋转

/* 切换正负值，观察旋转方向 */
transform: rotateZ(45deg);
/* transform: rotateZ(-45deg); */

代码演示-自定义轴旋转

.parent:hover .child {
  /* 沿X轴旋转 */
  /* transform: rotate3d(1, 0, 0, 45deg); */
  /* 沿Y轴旋转 */
  /* transform: rotate3d(0, 1, 0, 45deg); */

  /* 沿自定义轴，平面上的45度线旋转 */
  transform: rotate3d(1, 1, 0, 45deg);
}
